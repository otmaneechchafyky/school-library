# 📖 [School Library] <a name="about-project"></a>

**[School Library]** School Library App is a beautifully designed tool that streamlines the process of adding students, teachers, and books. It efficiently records and manages borrowing history, enhancing the library's organization and accessibility. A user-friendly interface ensures a seamless experience for librarians and users alike.
